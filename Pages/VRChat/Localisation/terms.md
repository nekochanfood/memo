## Boop(ing)

フレンドに絵文字を送る機能。Facebook にある Poke に近いもの。

日本圏では、別のインスタンスにいるフレンドにメッセージを伝えるために使われることがあるらしい。

"Boop(ing)" は「つっつく」とし、動詞では「つっつく」、名詞では「つっつき」とする。

```
EN: Boop X
JA: 「X」をつっつく

EN: Boop sent to X
JA: 「X」をつっつきました

EN: Send and Receive Boops
JA: つっつきを送受信する
```

## Claim(ing)

請求、主張を意味する言葉。~~苦情ではない~~

VRChat ではバンドルの文脈で使われる。

"Claim(ing)"は「受け取る」とする。

```
EN: Claim Bundle!
JA: バンドルを受け取る！

EN: You have a bundle available to claim!
JA: 受け取れるバンドルがあります！

EN: Claiming Bundle...
JA: バンドルを受け取っています...
```

## Launch Pad

クイックメニューを開くとまず出てくる画面のこと。

メインメニューにも Launch Pad メニューがあったが、2025.4.1 で [Live Now](#live-now) に置き換わった。

"Launch Pad"は「Launch Pad」とする。

```
EN: Launch Pad
JA: Launch Pad
```

## Live Now

以前メインメニューにあった [Launch Pad](#launch-pad) の進化版。

2025.4.1 で登場。A/B テスト機能であり、2026.1.1 で全ユーザーに解放された。

ページ上部には開催中のイベントが表示され、その下にはフレンドが居るインスタンスや、参加しているグループが持つインスタンスが表示される。
さらにその下には様々なジャンルに分けられたグループのイベントが表示される。

"Live Now"は「Live Now」とする。

無理に翻訳してぎこちなさが生まれてしまうより、英語のままにして「そういう名前の機能」という認識をしてもらうことにした。

```
EN: Live Now
JA: Live Now

EN: Double click to view the Live Now page.
JA: ダブルクリックで Live Now ページを開きます。
```

参考: なし


## Mutual

共通、相互を意味する言葉。

2025.4.2 でMutual 機能が登場したことにより用語の一つとなった。この機能を使うことで、ユーザーと共通のフレンドやグループを見ることができる。

"Mutual"は、単体では「共通」または「共通のつながり」とし、特定の単語と併せて使われる場合では「共通のXXX」とする。

```
EN: Mutuals
JA: 共通のつながり

EN: Mutual Discovery enabled/disabled
JA: 共通のつながりの発見が有効/無効になりました

EN: View mutual friends
JA: 共通のフレンドを見る
```

参考: なし


## Product(s)

商品、製品を意味する言葉。

VRChat では クリエイターエコノミ (CE) を通して購入者やサブスクライバーに提供されるものを指す言葉として使われる。

"Product(s)"は「商品」とする。

初期段階では「プロダクト」と翻訳していたが、後に変更された。
売り手向けのコンテキストで翻訳するなら「製品」が適切だが、買い手向けの場合は「商品」になるという特徴がある。そのため、場合によっては「製品」と訳される可能性がある。（e.g., CE の管理画面のテキスト）

```
EN: Subscribe to this product.
JA: この商品をサブスクライブする

EN: View more details about this product
JA: この商品の詳細を見る

EN: Contains the following products:
JA: 含まれる商品の一覧:
```

参考: https://discord.com/channels/1096246414603997194/1375303765770633226/1375316724945125497


## Pronouns

ここでの "Pronouns" は「ジェンダー代名詞」のことであり、自分が他社にどう呼ばれたいかを表明するもの。

日本圏では、第二のステータスとして使われたり、発音記号を用いたり (nekotʃan fúːd) 違う言語 (ねこちゃんふーど) での名前の呼び方を表示したりなどの使い方が見られた。

"Pronouns" は「代名詞」とする。

```
EN: Pronouns
JA: 代名詞

EN: Pronouns Updated
JA: 代名詞を更新しました

EN: Pronouns let others know how to address you in conversations.
JA: 代名詞は、他の人が会話の中であなたをどう呼ぶかの参考になります。
```

参考: https://discord.com/channels/1096246414603997194/1368046334556377219
